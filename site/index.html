Get started in 60 seconds

Completely free. No signup. No limits.

pip install xbridge-mcp

# Set your key
export XAI_API_KEY=your_key_here

# Run
xbridge-mcp

Or use Docker:
docker run -e XAI_API_KEY=xxx hrco/xbridge-mcp:latest
