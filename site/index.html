Architecture

How It Works

MCP (Model Context Protocol) is a standard that lets AI assistants use external tools. xBridge plugs Grok's API into Claude Code — like USB for AI.

Claude Code → MCP Protocol → xBridge MCP → xAI Grok API

You ask Claude. Claude decides it needs web data. It calls Grok through xBridge. Answer comes back. You never leave your terminal.

Questions

FAQ

Is this made by xAI?

No. xBridge MCP is an independent open-source project. Not affiliated with or endorsed by xAI.

Is it safe? Where does my API key go?

Your xAI API key stays on your machine — xBridge never proxies or logs it. The source is MIT licensed — inspect it yourself.

Can Claude set it up for me?

Yes. Claude Code understands MCP configuration. Give it the JSON config from the guide and it will add the server to your config file automatically.

What if I don't have an xAI API key?

Get one free at x.ai/api. xAI gives new users free credits to start.

Do I need $XBRDG to use xBridge?

No. $XBRDG is an optional community token that gives loyalty perks. xBridge works perfectly without it.
