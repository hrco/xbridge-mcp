# Launch Checklist (Lean, 72-hour)

## Day 1 — Payment + Offer
- [ ] Create product in LemonSqueezy or Stripe Payment Links
- [ ] Price at **€3.69/month**
- [ ] Add checkout URL to README (`https://YOUR-CHECKOUT-LINK`)
- [ ] Prepare private onboarding doc for buyers

## Day 2 — Distribution
- [ ] Push Docker image `hrco/xbridge-mcp:latest`
- [ ] Verify clean install path on a fresh machine
- [ ] Add "Buy Pro" section to repo and docs

## Day 3 — Go-to-market
- [ ] Publish launch post (X + Reddit + Discord communities)
- [ ] DM 10 warm dev contacts (short, no spam)
- [ ] Ask first 3 buyers for testimonial
- [ ] Track conversion in `docs/monetization/metrics.csv`

## Minimum Success Metrics (first 14 days)
- 100 repo visitors
- 10 checkout clicks
- 2 paid subscribers

## If Conversions Are Low
- tighten promise: "setup in <10 min"
- add quickstart GIF/video
- add one compelling case study
