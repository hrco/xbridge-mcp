Tailscale — VPS Setup

1. Install Tailscale on VPS 
curl -fsSL https://tailscale.com/install.sh | sh
2. Authenticate & advertise subnet (so you can reach VPS services from any device) 
sudo tailscale up --advertise-routes=76.13.48.186/32 --ssh

This makes the VPS join your Tailnet. The --ssh flag enables Tailscale SSH (no more SSH keys). Approve the device in the admin console at login.tailscale.com/admin/machines.

3. Enable Tailscale Serve (expose xbridge HTTP server on your Tailnet) 
sudo tailscale serve --bg 8000

Now your backend is at https://[vps-hostname].tailnet-name.ts.net — accessible only to your Tailnet devices. No public exposure, no DNS needed.

4. (Optional) Expose publicly via Funnel 
sudo tailscale funnel --bg 8000

This gives you a public HTTPS URL. Useful for webhooks (LemonSqueezy needs a public endpoint). Replaces the nginx/Traefik SSL setup if you want simplicity.

5. Install Tailscale on your local machine 
# macOS
brew install tailscale && sudo tailscale up

# Linux
curl -fsSL https://tailscale.com/install.sh | sh && sudo tailscale up

# Windows
winget install tailscale

Once connected, access VPS services via Tailscale IP (100.x.x.x) or MagicDNS hostname.
